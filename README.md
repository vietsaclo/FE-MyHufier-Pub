## How to run ?
1. `git clone -b develop https://github.com/vietnq-dgh/FE-MyHufier.git`
2. `cd FE-MyHufier/`
3. `yarn`
4. `yarn start`
## Requirements ?
- npm version 6.14.11 or higher `npm -v`
- yarn version 1.22.2 or higher `yarn -v`
- node version 12.0 or higher `node -v`
- ...
